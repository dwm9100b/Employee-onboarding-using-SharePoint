Now that our sites are created. We need to start adding our lists to the sites.

We will first work with the HR site.

  1. Navigate to the HR site created earlier, you'll see a link in top nav
  2. Top nav, click on +New, select List
  3. Create a new Blank list
  4. Name: WFOnboardingTasks
    1. Yes, create without spaces, you can change this later
    2. WF standing for a Workflow list
    3. Leave the check on Show in site navigation
  5. Click on Create
  6. Once created, go into Settings (click on upper right cog wheel)
  7. Click on List settings
  8. Turn off attachments, under advanced settings, if you don't want attachments
  9. Click on OK
  10. Back to the list
