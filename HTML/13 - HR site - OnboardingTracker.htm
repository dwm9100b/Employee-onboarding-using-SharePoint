Our onboarding tracker will also reside on the HR site.
The content of this list will be generated by a flow.

  1. In the left nav, click on home for the HR site
  2. Top nav, click on +New, select List
  3. Create a new Blank list
  4. Name: OnboardingTracker
  5. Click on Create
  6. Once created, go into Settings (click on upper right cog wheel)
  7. Click on List settings
  8. Turn off attachments, under advanced settings, if you don't want attachments
  9. Click on OK
  10. Back to the list

  11. Add columns as follows:

    Col Type Name Options Default Comments
    Text Title Always in every list regardless
    Text NewHire
    Date StartDate Date only, no time
    Number DaysToComplete
    Calculated DueDate =[StartDate]+[DaysToComplete]
    Returns DateTime, Date only
    Person AssignedTo Groups allowed
    Choice Status Pending
    Complete    		 Pending
    Hyperlink LinkToOnboardingInputForm A link back to the originating onboarding form
