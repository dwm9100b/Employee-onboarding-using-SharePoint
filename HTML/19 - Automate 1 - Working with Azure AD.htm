Let's continue working on our first automation.
If you are not already there, go to Power Automate.

Navigate to Solutions in the left nav, click on your New Hire Onboarding solution.

Edit the flow: When a new employee onboarding input form is submitted.

In this section, we will be working with Azure Active Directory.

  1. Add a new step after your variable initializing steps
  2. Search for and select: Create User (Azure AD)
  3. Rename to: Create user in AD
    1. Account enabled: Yes
    2. Display Name: {FirstName} {LastName} from dynamic content
    3. Mail Nickname: {varUserName}
    4. Password: {varPassword}
    5. User Principal Name: {varUserName}@your_domain_name.com
    6. Given Name: {FirstName}
    7. Surname: {LastName}
  4. We need to add in a delay so AD can catch up
  5. Add a new step
  6. Search for and select: Delay
  7. Rename to: Delay 2 minutes for cloud system catchup
    1. Count: 2
    2. Unit: Minute
  8. Add another new step
  9. Search for and select: Add user to group (Azure AD)
  10. Rename to: Add user to licensing group
  11. Whoops... we need an ID. So open up a new browser tab
  12. Unless you saved this in a scratch location as noted earlier
  13. Get into the Azure AD admin portal
    1. Select Groups in the left nav
    2. Click on All groups
    3. Find your SG Licensing group
    4. Highlight the Object Id and copy this
    5. Return to your flow tab
    6. Paste the copied Object Id into the Group id field
  14. For the User Id
    1. Select {ID} from dynamic content under the Create user section
  15. Optional, you could add a new step for Update User in AD for expanded fields
  16. Good time to save flow again
