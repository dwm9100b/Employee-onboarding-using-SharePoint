We will assume you have access to a Microsoft 365 environment which also provides you with a license for Power Apps.

If you do not, or want a playground all to your own, then I suggest you sign up for a developer tenant as per below:

  1. Sign up for a free Microsoft 365 Developer instance
  2. This is a 90 day E5 with 25 user licenses
  3. It will auto renew every 90 days as long as there is activity

One quick note. We will / you should create all objects without spaces. This will make it easier when selecting and programmatically accessing objects. Once the objects are created, you can go back and edit the name to include spaces. Renaming does not change the system name of the created object it only changes the display name (visible).

For example:

Object Type Name with spaces Created name
Sites Onboarding Demo OnboardingDemo
Lists WF Onboarding Tasks WFOnboardingTasks
Columns Logical Owner LogicalOwner

Also, when you see {FieldName} in the text of these documents, this means you'd pull this from the dynamic content popups. Make note of this as you will see this a lot as you move thru the tutorial.
