Best practices should probably be using solutions for your automations. It makes it easier for ALM like source control, moving between environments, etc.

  1. In your browser, navigate to Power Automate
  2. If not automatically signed in, do so.
  3. Towards the upper right, change to your dev environment
  4. In the left nav, click on Solutions
  5. In the top nav, click on + New solution
    1. Display name: New Hire Onboarding
    2. Name: NewHireOnboarding
    3. Publisher: If you have not created your own org publisher, do so now and select
    4. Version: Leave as default 1.0.0.0
  6. Click on Create
