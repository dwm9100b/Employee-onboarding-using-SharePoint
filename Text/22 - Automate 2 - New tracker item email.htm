Another automation, this time for our Onboarding Tracker.
If you are not already there, go to Power Automate.

Navigate to Solutions in the left nav, click on your New Hire Onboarding solution.

  1. Click on New, Automation, Cloud flow, Automated
    1. Name: Send email of onboarding task being assigned
    2. Trigger: When an item is created (SharePoint)
    3. Click on Create
    4. Site: /sites/HR
    5. List: OnboardingTracker
  2. Rename the first step to: When an item is created in onboarding tracker list
  3. Add a new step
  4. Search for and select: Get My Profile (V2)
  5. Add another new step
  6. Search for and select: Send an Email (V2)
    1. To: {AssignedTo Email} from dynamic content
    2. Subject: New Hire Task assigned, due: {DueDate} (from dynamic content)
    3. Body:

      Hello {AssignedTo DisplayName},

      You need to: {Title} for {NewHire} by {DueDate}

      Link to item: {Link To Item}

      Please update status to Complete when task is done

  7. This completes this automation.
    Make sure to save then return to Automate home
