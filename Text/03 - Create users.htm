This step is in a different order than the demo video.

We will be creating some users so that we can more easily follow the flow of the onbording process. If you don't have extra licenses then you can use any of your exiting users or you might want to get a dev subscription so you can do this without worry.

You'll be creating these users in Azure Active Directory.

  1. Open up Microsoft 365 admin center website
  2. From the Microsoft 365 admin center, click on show all in the left nav
  3. Under Admin centers, click on Azure Active Directory
  4. From Azure Active Directory, expand Users in the left nav
  5. Click on All users
  6. You'll be using the top nav +New user for each of the following
  7. Create the password and write it down so you can log into the accounts when needed
  8. Also, very important, for each user set the Usage location to your location
  9. If you don't set the location, you won't be able to assign licenses
  10. Now on you go to create each user:
User Name Name First Last Usage Location Title Dept
hr.mgr Hr Manager Hr Manager United States HR Manager HR
hiring.mgr Hiring Manager Hiring Manager United States Sales Manager Sales
facility.mgr Facility Manager Facility Manager United States Facility Manager Facilities
ops.mgr Operation Manager Operation Manager United States Operations Manager Operations

We now need to assign Microsoft 365 E5 licenses to each user created. Refresh the user list until all that you created show up.

  1. For each account you've created, click on them
  2. Then in the left nav, click on Licenses
  3. Click on + Assignments in the top nav
  4. Select the Microsoft 365 E5 Developer, leave all individual licenses selected
  5. Click on Save
  6. Do again until all 4 have been assigned their E5 licenses
  7. If you did not set the Usage location for any user, the license assignment will fail.
