Top site navigation on the hub site does not automatically get updated when sites are added to the hub. You'll have to update this manually.

  1. Navigate back to your Hub site (should see a link upper left)
    or go to /sites/OnboardingHub
  2. Very Top nav you'll see the name of your site with an Add link (or Edit) next to it
  3. Click the Add link, this will open the Edit hub navigation panel
  4. Click the + in the horizontal bar
  5. Select Link from the dropdown
  6. Enter the URL for the employee site (/sites/employees)
  7. Enter the display name you'd like to use (Employees)
  8. Put a check on the Open in new tab
  9. Press Ok
  10. Click the + in the horizontal bar again
  11. Select Link from the dropdown
  12. Enter the URL for the HR site (/sites/hr)
  13. Enter the display name you'd like to use (HR)
  14. Put a check on the Open in new tab
  15. Press Ok
  16. Once satisfied, click on the Save button
