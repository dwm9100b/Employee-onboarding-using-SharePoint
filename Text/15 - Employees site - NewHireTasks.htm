Next up, we will create a task list for the new employees.

  1. Navigate to the Employees site via the top nav
  2. Once on the site, top nav, click on +New, select List
  3. Create a new Blank list
  4. Name: NewHireTasks
  5. Click on Create

  6. Add columns as follows:

    Col Type Name Description Options Default Comments
    Text Title Task name Always in every list regardless
    Person AssignedTo New Hire, no groups
    Person Manager Profile pic enabled
    Number DaysToComplete
    Date StartDate Date only, no time
    Calculated DueDate =[StartDate]+[DaysToComplete]
    Returns DateTime, Date only
    Multiline Text Description
    Choice Status Pending
    Complete    		 Pending

  7. Let's create couple of views
  8. Go into list settings (upper right cog wheel)
  9. Under views, create View
  10. Click on All items as your starting view
  11. Name: MyItems
  12. Uncheck StartDate and DaysToComplete as they are not relevant
  13. Move down to the Filter section
  14. Click on Show items only when the following is true
  15. Set the filter to be:
    1. AssignedTo
    2. Is equal to
    3. [Me]
  16. Press Ok
  17. Back to List settings (remember, upper right cog wheel)
  18. Create View
  19. Click on MyItems as your starting view
  20. Name: Manager
  21. Scroll down to the filter section and edit as follows:
    1. Manager
    2. Is equal to
    3. [Me]
  22. Press Ok
  23. Back to List settings
  24. Turn off attachments, under advanced settings, if you don't want them on the form
  25. Back to the list
  26. On the title column dropdown, Select Column settings then Format
  27. Click on conditional formatting
  28. Edit the existing rule (click on the ... then edit)
    1. If DueDate is before Today
    2. Under Show list item as, click on the pencil
    3. Set color as red
  29. Press Save
  30. Click on Close to exit the panel
  31. Click on Views dropdown
  32. Select MyItems to make this view current
  33. Set MyItems as default view by clicking on the View dropdown
  34. Go back to list settings (upper right cog wheel)
  35. Click on Permissions
  36. Click on Advanced permissions settings
  37. Click on Stop inheriting permissions towards the upper left
  38. Select (put a check mark) on Employees Visitors
  39. Click on Edit User Permissions
  40. Uncheck the Read permissions
  41. Check the Contribute permissions
  42. Press Ok
  43. Done... go back to Employees Site
