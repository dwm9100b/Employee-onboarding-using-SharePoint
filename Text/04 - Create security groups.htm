This step is in a different order than the demo video. I'm making this step earlier so we can streamline the process and make it a bit more logical.

We will be using these security groups later.

  1. Open up Microsoft 365 admin center website
  2. Expand Teams & Groups
  3. Select Active teams and groups
  4. You will click on Add a group for each item in the following table:

We now need to go into Azure Active Directory to work on our SG licensing group.

  1. In the left nav, expand Show all
  2. Under Admin centers, click on Azure Active Directory
  3. Expand Groups
  4. Click on All groups
  5. Click on the SG Licensing group we just created
  6. On the overview page, copy the Object Id to a text editor for later
  7. Click on Licenses in the left nav
  8. Click on Assignments at the top nav
  9. Select the E5 license
  10. Decide which E5 products to add, for us just leave all selected
  11. Click Save
  12. Once saved, click the upper right X to close assignments
  13. You can close the browser tab for the Azure AD admin center
