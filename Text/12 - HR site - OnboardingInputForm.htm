Our onboarding list and form will reside on the HR site.

  1. In the left nav, click on home for the HR site
  2. Top nav, click on +New, select List
  3. Create a new Blank list
  4. Name: OnboardingInputForm
  5. Click on Create

  6. Add columns as follows:

    Col Type Name Description Options Default Comments
    Text Title Always in every list regardless
    Text FirstName
    Text LastName
    Date StartDate Only date, no time
    Choice WorkLocation HQ
    Remote
    Hybrid
    Choice OnboardType W2
    Contract
    Person HiringManager No groups allowed
    Text PersonalEmail Offsite personal email of the new employee

  7. Go into Settings (click on upper right cog wheel)
  8. Click on List settings
  9. Click on Title, under Require, change this to No
  10. Click on OK
  11. Turn off attachments, under advanced settings, if you don't want this on the form
  12. Click on OK
  13. Back to the list
  14. Click on the dropdown of any column, Title
    1. Column settings
    2. Show/hide columns
    3. Uncheck Title
    4. This will remove Title from the list view
    5. Click Apply (upper top of this panel)
  15. Create a new item for this list, we'll use this to edit the form only
  16. In the upper right of the form there is an edit form button, press this then Edit columns
  17. In the Edit Columns, uncheck the Title and rearrange the form items to your liking
  18. Click Save (upper part of this panel)
  19. Cancel the new item as we won't be using it
